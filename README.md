# factory_images
Работа с изображениямиями по конвертации и маштабированию. Основан по принципу микросервиса. В качестве примера используется реализация конвертиции изображения в формат WEBP с изменением размера. Принцип фабрики объектов с возможностью расширения других форматов для конвертации по аналогичному примеру. 

Пример использования: 

// создаем новое изображение из формы, где (обьект изображения, название диска куда сохранить, формат для сохранения) 
$convertImg = ImageFactory::buildImg($request['img'],'article','webp');

// масшабирование изображения и получаем путь к файлу
$pathImg = $convertImg->saveNewImg(350,350);
